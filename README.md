# Advanced-Programming
# Programming Assignments Repository

This repository contains solutions for 19 assignments covering:

* Time and Space Complexity Analysis (C)
* Java Collections Framework
* Python Data Structures
* React and React Native Development
* Object-Oriented Programming
* SOLID Design Principles
* Memory Management
* Multithreading and Synchronization
* Exception Handling and Unit Testing

## Repository Structure

Each assignment is maintained in a separate branch:

* Assignment-1
* Assignment-2
* Assignment-3
* ...
* Assignment-19

## Branch Contents

Every assignment branch contains:

* Source code implementation
* Supporting files (if applicable)
* README.md containing the original assignment question
* Complexity analysis where required

## Technologies Used

* C
* Java
* Python
* React.js
* React Native
* POSIX Threads (pthreads)
* Pytest
* JUnit 5

## Notes

Dependency folders and generated files such as node_modules, IDE configurations, build artifacts, and cache files have been excluded from version control using .gitignore to maintain a clean repository structure.
