# Assignment 19

## Question

Build a single-screen mobile application using React Native: a digital counter with theme toggling.

Features:
- Increment, Decrement, and Reset counter (counter never goes below 0)
- Theme Toggle button switching between Light Mode (white background, dark text) and Dark Mode (dark background, white text)

Implementation Rules:
1. UI Layout: Use View, Text, and TouchableOpacity. Center the counter UI using Flexbox.
2. State Management: Use useState for counter value and active theme mode.
3. Counter Logic: Start at 0; increment by 1; decrement by 1 but never below 0; reset to 0.
4. Dynamic Styling: Use conditional styling or ternary operators based on theme state.
5. Code Cleanliness: Use meaningful names (e.g., handleIncrement, toggleTheme).

Those with Android must run it on mobile in development mode. Those with iPhone may run it in Android Studio emulator.
